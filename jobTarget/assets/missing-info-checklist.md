# Missing Input Checklist

When inputs are missing, ask only for the smallest set needed to improve judgment.

## Core missing items

- JD or role summary
- candidate resume
- company and team name
- candidate priorities

## Suggested follow-up prompts

- 请提供该岗位的 JD，或至少贴出岗位职责和要求的核心部分。
- 请提供你的简历，或概述最近 2 到 3 段最相关经历。
- 请说明目标公司的名称、业务线或团队名称；如果不清楚，也请提供招聘页面或面试上下文。
- 请说明你最关注的因素，例如薪资、成长、稳定性、title、技术栈、出海机会、工作强度等。

## Optional but high-value inputs

- current compensation baseline
- competing opportunities
- recruiter notes
- manager or team background
- links to public posts or interview reviews
